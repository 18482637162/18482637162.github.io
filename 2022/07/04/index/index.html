下面介绍本人搭建博客的大致过程（环境为win10）（目前正在整理）

1、环境准备

  • nodeJS环境和git环境

    nodeJS和git直接百度搜索下载，一键傻瓜式安装即可

安装好nodeJS之后可以在cmd命令行使用 node -v 查看node版本，如果有显示版本号即表示安装成功

git则使用命令 git –version 进行查看

  • hexo框架安装

nodeJS安装好之后，已经附带安装好了npm(node package manager),然后使用下列命令安装hexo框架
